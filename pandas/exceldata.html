Pandas Project: Reading Excel Data

This project demonstrates how to use the pandas library to import and explore Excel data using a real estate dataset from Assam.

Function: Load and Describe Excel Data

This function loads the Excel sheet, prints preview data, and gives structure and statistics of the data.

Function Code


def explore_excel_data(file_path):
    import pandas as pd
    pd.set_option('display.max_columns', None)

    data = pd.read_excel(
        file_path,
        sheet_name="Sheet1",
        nrows=20,
        usecols="A:C",
        skiprows=2
    )

    print("First 4 rows:")
    print(data.head(4))

    print("Last 5 rows:")
    print(data.tail())

    print("Data information:")
    print(data.info())

    print("Data description:")
    print(data.describe())

    print("Index:")
    print(data.index)

    return data

Commentary

This project focuses on using the pandas library to explore Excel data. The function begins by configuring pandas to display all columns of the dataset without truncation. It then loads a specific sheet named "Sheet1" from the Excel file, restricting to the first 20 rows and columns A to C only, while skipping the first two header rows.

The output includes the first four and the last five rows to give a preview of how the data looks at both ends. Then, it uses info() to display metadata like column names, non-null entries, and data types. describe() provides a statistical summary of numeric fields or unique/object statistics where applicable. Finally, data.index shows the index range of the dataset.

The function returns the DataFrame so it can be used for further analysis or visualizations in other parts of the project.
