This project demonstrates how to use the pandas library to import and explore Excel data using a real estate dataset from Assam.

üí° Function: Load & Describe Excel Data

This function loads the Excel sheet, prints preview data, and gives structure and statistics.

üíª Function Code


def explore_excel_data(file_path):
    import pandas as pd
    pd.set_option('display.max_columns', None)

    data = pd.read_excel(
        file_path,
        sheet_name="Sheet1",
        nrows=20,
        usecols="A:C",
        skiprows=2
    )

    print("First 4 rows:")
    print(data.head(4))

    print("Last 5 rows:")
    print(data.tail())

    print("Data information:")
    print(data.info())

    print("Data description:")
    print(data.describe())

    print("Index:")
    print(data.index)

    return data

üîç Commentary

  • Function Name: explore_excel_data for clarity and reuse.
  • Input: Takes in the full path to your Excel file.
  • pd.set_option: Ensures all columns are shown when printed.
  • read_excel: Loads only the relevant portion (top 20 rows, columns A to C).
  • head/tail/info/describe: Used for quick inspection and summary.
  • Return: DataFrame is returned for further use in analysis/visualization.
